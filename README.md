# Hair by Tigi

A modern, responsive website for Hair by Tigi, built with Next.js and Tailwind CSS.

## Features

-   **Modern Design:** Clean, elegant aesthetic matching the "natural beauty" brand.
-   **Responsive:** Looks great on mobile, tablet, and desktop.
-   **Services Section:** Clear display of services offered.
-   **Information:** Easy access to business hours, location, and contact info.
-   **Booking Integration:** Direct link to Sola Salon Studios booking.

## Tech Stack

-   [Next.js](https://nextjs.org/)
-   [Tailwind CSS](https://tailwindcss.com/)
-   [pnpm](https://pnpm.io/)
-   [Lucide React](https://lucide.dev/) (Icons)

## Getting Started

First, install dependencies:

```bash
pnpm install
```

Then, run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Building for Production

To create a production build:

```bash
pnpm build
```

To start the production server:

```bash
pnpm start
```
